# Python Automation Scripts

## Description
This repository contains Python scripts for automating tasks like file management, AWS operations, and system monitoring.

## Prerequisites
- Python 3.x
- Required libraries (listed in requirements.txt)

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/python-automation.git
   cd python-automation
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Scripts Included
- **AWS Automation**: Automate EC2 and S3 operations.
- **System Monitoring**: Monitor CPU and memory usage.
- **File Management**: Organize and clean up directories.

## Usage
Run scripts using:
```bash
python script_name.py
```
